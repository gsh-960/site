# 六周年纪念

亲爱的字政榆：

见字如面，展信欢！

1月1日对我我们是个有多重含义的日子，是新一年的开始，也是我们的开始。

时序轮转，爱意绵长，往年的心动，仿佛还在昨日，转眼我们便有走过了一个四季的昏沉

提笔瞬间，诸多片段浮现心头，是每次见面的晚风，是爱琴海的楼顶，是元江的街头巷尾，是车站的迎接，也是车站的告别，只有我两能真正体会每个片段的快乐与辛酸，不管在什么时候，这些都是我两最宝贵的回忆。

提笔给你写这封信时，才惊觉这一年的时光里，藏着太多值得铭记的刻度。1月你上岸元江，结束了长达两年的就业焦虑，但是伴随而来又是体制内无聊无尽的职场生活，就如同我上班第一年，几乎磨灭了所有的工作热情，我想说的是，先别放弃，每一份工作对新人都不友好，试着在第二年里找到这份工作的价值，我会一直给你决策上的支持;5月你带我见了你爸妈，我们一起吃饭聊天，云南省内家长对于体制内的执着又再一次震惊到我，但别灰心，我有信心在我的职位上取得家长们的认同，我对自己有信心，你也要对我有信心;6月，我甲流严重，一个人躺倒在出租屋里，记得是6月21日晚上我两还发生了严重的争吵，病痛加上吵架，让我身心俱疲，删除了我们所有的联系方式，一度想结束这段关系，但冷静下来后感到后怕，又赶紧找回了联系方式，试图修复关系，让我没想到的第二天你就从元江到了佛山，出现在了我的门口，我到现在都记得你疲惫的模样，你的勇敢拯救了我们的爱情，哈哈哈；8月，你送了最好的礼物，IntelliJ 全家桶的会员，是个Java程序员都知道这份礼物的含金量；9月，第一次因为给你找礼物焦虑了一个月，好在最后我两都比较满意；10月，你第一次说服你爸妈，到佛山旅游，这次我们没有穷游，去了很多地方，珠海，桂山岛，深圳华强北，水贝，广州，这让我回想到大学时期偷偷穷游的日子，不同时期有不同时期的开心。

每次回忆与你在一起的片段都非常开心，不知不觉我们已经一起走过了六年，我们已经从高中生变成了辛苦的大人，两次毕业，我们都做了不同的选择，都发生了不同的争吵，都为未来付出了努力。回看过去的一些选择，有对有错，见证了我们的成长，但我觉得我们的方向都没错，我坚信我们会越走越近，最后永远在一起。

今年，我想给你分享我的成长，在你的影响下，我学会了你的共情能力，学会了关注别人的情绪，学会了怎么闲聊，学会了明天的事情明天去做，慢慢的，我变得不再焦虑，关注了更多代码之外的生活；此外，我更关注了自己的生活，我记住了更多人的生日，记住了更多的节日，每个有意义的日子我都会认真准备，让生活有更多的仪式感。我也关注到了你的成长，你不再把所有注意力放在自己的情绪感受上，逃离了情绪价值陷阱，你试着接受现状，适应不同性格的同事领导，减少了负面情绪的输出，有意识的锻炼身体，明天的事情明天再做，不再焦虑，我觉得“明天的事情明天去做”，这是我两今年共同取得的最大的进步，我们有了更多的耐心，能把一件事坚持得更久，这对于我两的现状是最有意义的。

不知不觉，已经写了一个小时，我现在的文字功底没办法概括我们这一年的点滴，得益于我们每晚视频的习惯，虽然异地，但也基本知悉对方的生活，碎念一些琐事就已经占满了大半的篇幅，但每次表达这些，不管是用文字还是视频，我都觉得心情愉悦。

最后我想画一下明年的饼，明年是我职业生涯的第三年，是第一个节点，我会尽量寻找拉近物理距离的机会，争取缩短一下每次见面的时间，你可以保持期待，但一定要保持平常心来对待这件事，因为我目前在公司还有很多机会和进步空间，我希望以更好的姿态回去，而不是将就现状，这是对自己负责，也是给你爸妈的交代，我相信你能理解。不管怎么样，明年年中我一定会交出一份答卷。

不管怎么样，我们都要为以后的生活努力，要再后面漫长的生活中互相扶持，互相理解，互相爱着对方，原我们岁岁年年，朝朝暮暮，爱与时光，皆不相负。
